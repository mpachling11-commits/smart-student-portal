Database access

MongoDB collections used by this website

The backend uses the database configured in .env. Default local database: student_placement_portal.

  1. Start MongoDB locally or paste your MongoDB Atlas URI into .env.
  2. Run npm run seed to create sample students and jobs.
  3. Run npm start and open http://localhost:3000.
  4. Open MongoDB Compass, connect to mongodb://127.0.0.1:27017, then browse student_placement_portal.
  5. Use collections students, jobs, and applications to view live website data.
